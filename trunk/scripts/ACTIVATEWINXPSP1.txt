How to active Windows SP1 (or later)

1. Save below ChangeVLKey.vbs to disk and run it

' Save this content to ChangeVLKey.vbs
ON ERROR RESUME NEXT
 
Dim VOL_PROD_KEY
VOL_PROD_KEY = "WM987-9RRB9-PY7GG-BT9GJ-224XT"
VOL_PROD_KEY = Replace(VOL_PROD_KEY,"-","") 'remove hyphens if any

for each Obj in GetObject("winmgmts:{impersonationLevel=impersonate}").InstancesOf ("win32_WindowsProductActivation")
 
result = Obj.SetProductKey (VOL_PROD_KEY)
 
if err <> 0 then
WScript.Echo Err.Description, "0x" & Hex(Err.Number)
Err.Clear
end if
 
Next
' End of ChangeVLKey.vbs

2. Open the file C:\WINDOWS\system32\drivers\etc\hosts and add this line: 127.0.0.1 mpa.one.microsoft.com
3. Run MGADiag.exe from http://go.microsoft.com/fwlink/?linkid=52012  to active Windows XP
